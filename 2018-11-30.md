# Today I Learned  
## android에서는 문제없이 잘 돌아가는 기능이 ios에서는 제대로 작동하지 않는 경우가 있음.  
  - 모바일 스크롤  
    - overflow  
    ```scss  
      -webkit-overflow-scrolling: touch;  
    ```  
    이 코드 overflow가 설정된 요소의 스타일 블럭에 추가하면 ios의 모바일 화면에서 스크롤이 매끄럽게 작동하게 됨.
